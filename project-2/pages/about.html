Why I chose this tutorial

I needed a way to install an AMD Vega FE card along side my Nvidia M4000 in my personal workstation and found that the supplied script would not work since it was installing several X11 components that conflicted, even with the --headless argument provided. I wanted to use the card as an OpenCL Device only, without any display capabilities. There is very little information on this subject, so I wanted to share how I eventually accomplished this unique setup.

In addition to using an AMD card along-side a Nvidia card, other circumstances might benefit from this setup, including headless compute servers that might want to minimize complications arising from the additional bloat that the install script provides.

As far as I know there is no further information on this specific topic.
